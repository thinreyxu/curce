The url.parse() and url.stringify() Demo

  • To parse an url string, input it into the "Url string" textarea, and then click on the "Parse" button.
  • To stringify an url object, input the string representation of the url object (in JSON format) into the "Url object" textarea, and then click on the "Stringify" button.

Url string

Sample

http://thinreyxu.github.io:80/curce/demo/url/parse&stringify/parse&stringify.html?id=curce&t=3dne83hf#page1

Url object

Sample

{
  "protocol": "http:",
  "hostname": "thinreyxu.github.io",
  "port": "80",
  "pathname": "/curce/demo/url/parse&stringify/parse&stringify.html/",
  "search": "?id=curce&t=3dne83hf",
  "hash": "#page1"
}
